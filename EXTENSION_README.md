Nova Syntax Extension
=====================

This small extension provides basic syntax highlighting for files with the `.nova` extension in Panic Nova using a TextMate grammar.

Installation
------------

- Zip the repository or copy the folder into Nova's Extensions directory.
- In Nova: choose `Extensions` → `Install Extension...` and select the folder or zip.

Files added
-----------

- `package.json` — extension manifest
- `Syntaxes/nova.tmLanguage.json` — TextMate grammar

You can expand the grammar in `Syntaxes/nova.tmLanguage.json` to add more patterns and scopes.
